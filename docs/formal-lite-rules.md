# Formal-Lite Rules for Learning and Future Backtest

## 目标

这份文档不是最终量化规则。
它的目标是两点：

1. 帮助学习者更清楚地理解 V1 / V2 在看什么
2. 为后续 1 年回测准备一版更明确的规则草稿

原则：
- 尽量保留原框架精神
- 减少模糊表达
- 暂不假装已经完成严格量化验证

---

## 五个核心观察对象

1. **Funding Rate (FR)**
   - 看拥挤度是否极端或中性

2. **Futures CVD**
   - 看期货主动买卖方向

3. **Spot CVD**
   - 看真实资金是否跟随

4. **Open Interest (OI)**
   - 看行情来自新开仓还是平仓

5. **Orderbook / Liquidation Context**
   - 看短线压力和潜在吸引区

---

## V1：逆向极值型（学习版）

### V1 的核心问题
不是问“价格是不是跌很多了”，而是问：

- 市场是否出现极端拥挤？
- 真实资金是否没有继续顺着恐慌跑？
- 当前走势更像杠杆挤压，而不是现货主导趋势？

### V1 做多，学习版条件
优先观察以下 4 条：

1. **FR 达到负向极值**
   - 当前参考：`FR <= -0.0040`
   - 备注：这是经验阈值，不代表最终回测阈值

2. **Futures CVD 继续走弱**
   - 表示期货端仍在恐慌或空头仍在加压

3. **Spot CVD 未继续明显恶化**
   - 允许仍为负
   - 但最好表现为：跌速放缓、走平，或开始修复

4. **OI 不支持“新一轮单边下跌”叙事**
   - 常见有利信号：OI 下降
   - 说明更像去杠杆，而不是大量新空继续压盘

### V1 做空，学习版镜像条件
与上面镜像：

1. `FR >= +0.0040`
2. Futures CVD 继续走强
3. Spot CVD 不跟涨，或开始转弱
4. OI 不支持“新一轮单边上涨”叙事

### V1 的学习重点
V1 不是“FR 极端就反手”。
V1 真正看的是：

**极端拥挤 + 现货不确认原方向 + 杠杆结构接近失衡。**

### V1 暂定判定方式
学习阶段可先用：
- 满足 4 条中的 3 条，记为 **V1 候选信号**
- 满足 4 条中的 4 条，记为 **V1 强信号**

这不是最终交易规则，只是为了统一观察口径。

---

## V2：趋势确认型（学习版）

### V2 的核心问题
不是问“是不是已经涨了”，而是问：

- 这波走势是不是有真实资金参与？
- 期货、现货、OI 是否在讲同一个故事？
- 当前更像趋势延续，而不是短线 squeeze？

### V2 前置条件
1. **FR 处于中性附近**
   - 当前参考：`abs(FR) <= 0.0010`
   - 含义：市场没有明显极端拥挤

2. **不存在明显现货/合约背离**
   - 如果 futures 很强、spot 很弱，优先降级为观望

### V2 做多，学习版打分

- Spot CVD 明显走强：`+3`
- OI 上升且价格走强：`+1`
- Orderbook 偏买方：`+1`
- Liquidation context 更利于向上推进：`+1 ~ +2`

### V2 做空，学习版打分

- Spot CVD 明显走弱：`-3`
- OI 上升且价格走弱：`-1`
- Orderbook 偏卖方：`-1`
- Liquidation context 更利于向下推进：`-1 ~ -2`

### V2 学习版结论
- `>= +5`：偏多趋势候选
- `<= -5`：偏空趋势候选
- `-4 ~ +4`：观望

### V2 的学习重点
V2 不是“所有指标都绿就追”。
V2 真正看的是：

**现货、期货、OI 是否形成同向共振。**

---

## 观望规则（非常重要）

学习阶段建议优先学会“不下结论”。

以下情况默认观望：

1. FR 不极端，也不中性清晰
2. Spot CVD 与 Futures CVD 明显冲突
3. OI 无法帮助解释当前走势
4. 宏观事件正在主导市场
5. 数据缺失或口径不稳

---

## 为 1 年回测预留的收紧方向

后面如果要回测，这份 learning rules 还需要继续收紧成正式规则，至少补齐：

1. **时间周期固定**
   - 例如全部使用 1h 观察，4h 做背景

2. **字段固定**
   - FR 用哪种口径
   - CVD 用哪种聚合
   - OI 用绝对值还是变化率

3. **入场规则固定**
   - 信号出现后立刻进，还是等回踩

4. **退出规则固定**
   - 固定止损
   - 固定止盈
   - 超时退出

5. **失效条件固定**
   - 哪种变化出现后，原信号作废

- 后续回测落地

如果后面做 1 年回测，完全可以以这份文档为起点继续收紧。
