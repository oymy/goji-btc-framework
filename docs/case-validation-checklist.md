# Case Validation Checklist

## 用途

这份表不是回测表。
它是“证明复现”用的核对表。

检查的是：
- 同一时间点
- 同一数据口径
- 是否能得出与原版一致的框架结论

---

## Case #001 2026-03-31 轧空教科书案例

### 原版已声明结论
- 这是轧空，不是真实趋势反转
- 不能直接追多
- 现货 CVD 转正才是更强确认

### 复现时必须核对
- FR 是否出现显著下降
- Futures CVD 是否上升
- Spot CVD 是否为负
- OI 是否下降
- 结论是否仍为：轧空，不等于趋势反转

### 判定通过标准
如果复现版不能得出“轧空而非真实买盘”，则不算一致。

---

## Case #002 2026-04-02 关税冲击日背离分析

### 原版已声明结论
- FR = `-0.0061`
- V1 信号触发
- 合约卖压远大于现货卖压
- 这是虚假跌，不追空
- 等清算密集区入场做多

### 复现时必须核对
- FR 是否达到原版阈值
- Futures CVD vs Spot CVD 是否出现极端背离
- OI 是否下降
- 结论是否为：V1 做多候选，而不是继续追空
- 是否保留“等待清算密集区，不追市价”

### 判定通过标准
如果复现版方向不是偏多候选，或忽略“等清算区”执行规则，则不算一致。

---

## Case #003 2026-04-01 FR 灰色地带看跌背离

### 原版已声明结论
- FR 在 V2 中性区间
- 合约 CVD 上升，现货 CVD 下降
- 这是看跌背离
- 切回 V1 观察模式
- 结论：观望

### 复现时必须核对
- FR 是否在 ±0.0010 内
- Futures CVD 与 Spot CVD 是否反向
- 系统是否拒绝直接给出 V2 做多
- 最终结论是否为观望

### 判定通过标准
如果复现版在这个案例上给出 V2 多头结论，则不算一致。

---

## Case #004 2026-04-02 V1 做多完整信号确认

### 原版已声明结论
四条件全部满足：
1. FR <= -0.0040
2. 合约 CVD 持续下降
3. 价格跌幅小于合约 CVD 跌幅
4. 现货 CVD 仍负但方向不恶化

执行：
- V1 做多成立
- 等 `$65,300-$66,000` 清算密集区入场

### 复现时必须核对
- 四条件是否能逐条还原
- 结论是否为 V1 做多成立
- 执行是否仍为“等区间，不追市价”

### 判定通过标准
如果四条件无法逐条映射，当前只能算近似理解，不能算证明复现。

---

## Case #005 2026-03-31 晚间 轧空 vs 真实买盘辨别法

### 原版已声明结论
- Spot CVD ↑ + OI ↑ = 真实买盘
- Spot CVD ↓/持平 + OI ↓ = 轧空

### 复现时必须核对
- 是否把 Spot CVD 放在比 Futures CVD 更高优先级
- 是否把 OI 变化当作“新资金 vs 平仓”的判别器
- 是否保留“轧空不等于趋势反转”的判断框架

### 判定通过标准
如果复现版只看价格，不看 Spot CVD / OI 组合，则不算一致。

---

## 总标准

要说“原版已正确复现”，至少要满足：
- 这 5 个案例里，核心方向判断基本一致
- 执行逻辑没有被偷偷改写
- 使用的数据口径不明显偏离原版

否则只能说：
- 学到了原版思想
- 做了近似实现
- 还不能说证明复现
