# Case Data Requirements

## 目标

列清楚每个案例要证明复现，至少缺哪些数据。

## 所有案例共通必需
- CoinGlass 同口径 funding
- CoinGlass 合约 CVD
- CoinGlass 现货 CVD
- OI 同时间点读数
- 清算热力图 / 清算密集区截图或结构化数据
- 对应时点价格

## Case #001
需要：
- 2026-03-31 对应时点 funding
- 现货 CVD 为负的证据
- OI 下降的证据

## Case #002
需要：
- 2026-04-02 关税冲击时点 FR = -0.0061 的证据
- 合约 CVD -$180M、现货 CVD -$15M 的同时间点证据
- 清算密集区 `$65,300-$66,000` 的来源证据

## Case #003
需要：
- 2026-04-01 FR = +0.0003 的证据
- 合约 CVD +$26.9M、现货 CVD -$2.2M 的证据

## Case #004
需要：
- 四条件同一时间点读数
- Higher Low 的价格结构证据
- 现货 CVD “不恶化”的原始快照

## Case #005
需要：
- Spot CVD / OI / Funding / 清算比 的对照证据
- 原版对“真实买盘 vs 轧空”的判断截图或原始记录
