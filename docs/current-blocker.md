# Current Blocker

## 现状

准备进入 B 档数据抓取时，当前机器直连交易所公开 API 出现阻塞：

- Binance: HTTP 451
- Bybit: HTTP 403
- OKX: HTTP 403

这说明当前问题不是规则没定义好，而是：

**回测原始数据现在拉不下来。**

## 影响

当前可以继续做：
- 文档
- 数据结构
- 抓取脚手架
- 回测脚本骨架

当前不能继续做：
- 真正抓到 1 年原始数据
- 跑 first-pass 事件研究
- 跑完整回测结果

## 最可能原因

1. 地区限制
2. 云/代理出口被交易所风控
3. 接口层反爬或 WAF 拦截

## 当前结论

项目没有逻辑性阻塞，
但有**环境级阻塞**。

在这个阻塞解除前，只能先把工程骨架做到位，不能产出真实回测结果。
