# Chapter 1: 框架原理 — 为什么这套逻辑有效

## 给不炒币的人看的版本

想象一个农贸市场。

你在市场门口观察：每天早上，一批买家进来购买苹果，另一批人在市场外通过"期货合约"赌苹果价格。

有一天，你注意到一个奇怪的现象：
- 市场里真实买苹果的人越来越少（现货 CVD 下降）
- 但场外赌苹果涨价的人越来越多（资金费率持续为正）

你会怎么判断？场外的赌注没有真实买盘支撑，迟早要崩。

**这就是 GojiPower 框架的核心逻辑。**

## 底层机制一：期现套利的自动修复力

### 资金费率是什么？

在 BTC 永续合约市场，多空双方每隔 8 小时互相支付一笔费用：
- 如果大多数人做多 → 资金费率为正 → 多头付费给空头
- 如果大多数人做空 → 资金费率为负 → 空头付费给多头

**关键洞察：** 资金费率是空头（或多头）的"持仓成本"。

### 为什么极端资金费率不可持续？

当资金费率达到极端负值（比如 -0.0040，即每 8 小时空头支付 0.4%）：

套利者会立刻行动：
```
同时买入现货 + 做空期货 = 锁定无风险收益
→ 现货买盘增加（推高现货价）
→ 期货卖盘增加（压低期货价）
→ 资金费率自动向 0 收敛
```

这不是预测，这是套利机制的必然结果。GojiPower 将这个收敛窗口定义为 **10 分钟执行窗口**。

**用人话说：** 当空头持仓成本贵到无法维持，他们必须平仓。平仓就是买入，买入就是拉价格。

## 底层机制二：CVD 背离揭示资金真实流向

### CVD（累计成交量差值）是什么？

CVD 测量的是：主动买入 vs 主动卖出的累计差值。

- 主动买入：以高于卖价成交（击穿卖单）= 有人迫切想买
- 主动卖出：以低于买价成交（击穿买单）= 有人迫切想卖

**CVD 是市场参与者"紧迫程度"的温度计。**

### 现货 CVD vs 合约 CVD：谁说的算？

| 来源 | 含义 | 可信度 |
|------|------|--------|
| 现货 CVD | 真实资金在购买/出售 BTC | 🔴 高（无杠杆，真实成本） |
| 合约 CVD | 杠杆资金在加注方向 | 🟡 中（有杠杆，可能是赌注） |

**核心原则（GojiPower 原话）：**
> "现货是真钱，合约是杠杆。现货不跟的上涨 = 虚涨 = 陷阱。"

### 三种关键组合解读：

**① 看涨背离（最强做多信号）**
```
现货 CVD ↗ + 合约 CVD ↘
→ 真钱在低价吸筹，期货空头在加仓
→ 空头付费成本上升 + 现货买盘存在 = 轧空条件成熟
```

**② 看跌背离（做多陷阱）**
```
现货 CVD ↘ + 合约 CVD ↗
→ 杠杆推价，真实买盘不跟
→ 一旦杠杆清算，价格失去支撑
→ 反弹是假的，不追
```

**③ 双向同步（趋势确认）**
```
现货 CVD ↗ + 合约 CVD ↗ + OI ↑
→ 新多头入场，趋势健康
→ V2 趋势框架适用
```

## 底层机制三：清算热力图的"引力"效应

杠杆交易的本质：在特定价位设置止损。

当大量止损集中在某个价位区间（热力图显示为高亮红柱），价格靠近时会产生两种效果：

1. **多头清算区（上方）= 阻力**：价格拉到这里触发止损，卖压激增
2. **空头清算区（下方）= 支撑**：价格跌到这里触发轧空，买压激增

**实战应用：** V1 信号触发后，不追市价，等价格回落到空头清算密集区再入场。那里有轧空的"地基"，胜率更高，止损更自然。

## 底层机制四：OI 揭示资金来源

未平仓合约（Open Interest）告诉你：是谁在推价格？

| 价格 ↑ + OI ↑ | 新多头入场（真实趋势） |
|---------------|----------------------|
| 价格 ↑ + OI ↓ | 空头平仓推价（轧空，趋势可疑） |
| 价格 ↓ + OI ↑ | 新空头入场（熊市确认） |
| 价格 ↓ + OI ↓ | 多头平仓（去杠杆，方向不明） |

轧空推出来的涨幅没有新买盘接力，通常在 OI 回升之前就反转。

## 五指标综合判断逻辑

GojiPower 框架用 5 个指标互相验证，避免单一信号的误判：

```
资金费率（FR）    → 判断多空拥挤程度
合约 CVD          → 判断期货方向意愿
现货 CVD          → 判断真实资金流向
OI                → 判断资金来源（新入场 vs 平仓）
订单簿深度差值    → 判断即时买卖压力
```

**3/5 规则**：5 个指标中 3 个以上一致，信号有效。不凑数，不强行判断。

## 这套框架什么时候最有效？

✅ **高效场景：**
- BTC 市场情绪极端（恐惧或贪婪指数在边界）
- 资金费率持续偏离中性超过 24 小时
- 价格与 CVD 出现明显背离
- 清算热力图有密集分布区在近价

❌ **失效场景：**
- 重大宏观事件（Trump 讲话、CPI 公布、战争事件）
- 假期/低流动性窗口（Good Friday、圣诞节、春节）
- 价格处于负 Gamma 区间（$68K 以下的 Deribit 期权格局）
- 所有指标混乱，无明确方向（此时框架答案是：观望）
