## 核心原则（必须遵守）

1. **全网聚合数据，绝无例外** — 永远使用 CoinGlass 全网 OI 加权数据，不用单所数据
2. **现货 CVD 是否决权** — 现货不认可的上涨是虚涨，不追
3. **3/5 规则** — 5 个指标中 3 个以上一致，信号才有效
4. **观望是有效选项** — 不确定时明确说「观望」，不强行判断
5. **10 分钟执行窗口** — 信号触发后 10 分钟内决定，错过等下一个

## 为什么全网 OI 加权资金费率如此重要？

**实证案例（2026-03-25 测量）：**
- Binance 单所资金费率：0.0022
- 全网 OI 加权资金费率：0.0043
- **差值：近一倍！**

V1 框架的阈值（±0.0040）是基于全网 OI 加权值设定的。如果用单所数据：
- 单所 FR 0.0022 < 阈值 0.0040 → 不触发 V1
- 但全网 FR 0.0043 > 阈值 0.0040 → 应该触发 V1

**用单所数据做 V1/V2 判断 = 拿错了尺子。**

### 没有 API key 的替代方案

**方案 A：多交易所加权估算**
```bash
# 获取 Binance、Bybit、OKX 资金费率，按 40%/30%/30% 加权
BINANCE_FR=$(curl -s "https://fapi.binance.com/fapi/v1/premiumIndex?symbol=BTCUSDT" | jq '.lastFundingRate | tonumber')
BYBIT_FR=$(curl -s "https://api.bybit.com/v5/market/funding/history?category=linear&symbol=BTCUSDT&limit=1" | jq '.result.list[0].fundingRate | tonumber')
OKX_FR=$(curl -s "https://www.okx.com/api/v5/public/funding-rate?instId=BTC-USDT-SWAP" | jq '.data[0].fundingRate | tonumber')

WEIGHTED_FR=$(echo "$BINANCE_FR * 0.4 + $BYBIT_FR * 0.3 + $OKX_FR * 0.3" | bc -l)
```

**方案 B：仅用 V2 框架**
V2 依赖 CVD 和 OI 方向，对 FR 绝对值要求较低。可以用单所 FR 判断「是否在 ±0.0010 内」。

**底线原则：** 数据不完整时，结论为「数据不完整，建议观望」，不编造数据。

## 五指标分析标准

```
资金费率（FR）     → OI加权全网，格式：-0.0038（不带%号）
合约 CVD          → Binance+Bybit+OKX 三所求和，单位 $M 或 $K
现货 CVD          → 四所求和（含Coinbase），同上单位
OI（未平仓合约）  → 全网聚合，USD计价（如 $48.2B）
订单簿深度差值    → Futures B&A Delta，单位 BTC（如 +1200 BTC）
```

**数据异常处理：**
- FR 超过 ±0.05% → 标注 [ANOMALY]，不使用
- 获取失败 → 报告标注「数据不完整，建议观望」

## V1 框架：逆向极值型

**适用场景：** 资金费率持续偏离中性（> 24 小时）+ 价格与 CVD 背离

**触发条件（严格）：**
- FR ≤ -0.0040（做多）或 FR ≥ +0.0040（做空）
- 必须用全网 OI 加权值
- 5 指标中 3 个以上一致

**V1 做多四条件验证：**
1. FR ≤ -0.0040（全网 OI 加权）
2. 合约 CVD 持续下降
3. 价格跌幅小于合约 CVD 跌幅（Higher Low 形成）
4. 现货 CVD 绝对值仍负但方向不恶化

**执行纪律：**
- 信号即时性：触发后 10 分钟内决定，错过等下一个
- 入场方式：等清算密集区，不追市价
- 止损位置：用结构高/低点，不用当根 K 线
- 仓位大小：宏观不确定时 50% 标准仓位试探
- 止盈方式：固定 TP（极值快速收敛）

## V2 框架：趋势确认型

**适用场景：** 资金费率在中性区间（±0.0010）+ 指标共振

**评分系统（-9 到 +9）：**

| 指标 | 权重 | 说明 |
|------|------|------|
| 现货 CVD | ±3 | 核心指标，真实资金方向 |
| OI | ±1 | 新资金入场确认 |
| 订单簿深度差值 | ±1 | 即时买卖压力 |
| 清算流动性 | ±2 | 上下方清算不对称性 |

**入场标准：**
- ≥ +5：做多
- ≤ -5：做空
- -4 到 +4：观望

**灰度仓位规则：** 评分 3-4（接近但未达标）+ 大结构完整 → 30-50% 试探仓位

**V1 vs V2 优先级：** 同时触发时 V1 优先。CVD 出现背离 → 强制切回 V1 观察模式。

---

## CVD 背离判断（不可混淆）

| 结构 | 含义 | 操作 |
|------|------|------|
| 现货CVD↗ + 合约CVD↘ | 看涨背离（真钱吸筹）| V1 做多条件之一 |
| 现货CVD↘ + 合约CVD↗ | 看跌背离（杠杆虚涨）| 切V1观察，不做多 |
| 双向同步↗ | 趋势确认 | V2 做多 |
| 双向同步↘ | 趋势确认 | V2 做空 |

**GojiPower 原话：** "现货是真钱，合约是杠杆。现货不跟的上涨 = 虚涨 = 陷阱。"

## 框架失效场景（明确观望）

1. **重大宏观催化剂** — Trump 讲话、地缘政治、央行紧急政策
2. **低流动性窗口** — Good Friday、圣诞节、春节（CME 关闭、ETF 申赎暂停）
3. **负 Gamma 区间** — 价格在 $68K 以下（Deribit put positioning 形成负 Gamma 反馈环）
4. **所有指标方向混乱** — 最好的交易有时候是不交易

**观望是有效选项。** 当框架无法给出明确方向时，「观望」是正确的输出。

## 真实案例参考

### 案例 #001：2026-03-31 轧空教科书

**背景：** BTC 价格 $66,760，市场恐惧情绪弥漫。

| 指标 | 数值 | 信号 |
|------|------|------|
| 资金费率 | +0.1731% → +0.0061%（暴降96%） | 多头去杠杆激烈 |
| 合约 CVD | ↑（轧空推动） | 空头被迫平仓 |
| 现货 CVD | -$83.5M（现货净卖出） | ⚠️ 现货不跟 |
| OI | -$1.2B | 空头出清，非新多入场 |

**判断：** 价格从 $66,760 涨至 $68,194（+2.14%），但现货 CVD 为负 → 真实买盘不跟。**轧空 ≠ 趋势反转。**

### 案例 #002：2026-04-02 关税冲击日

**背景：** Trump 宣布关税，BTC 与美股同步暴跌。

| 指标 | 数值 | 信号 |
|------|------|------|
| 资金费率 | -0.0061（超越 V1 阈值） | 🔴 V1 触发 |
| 合约 CVD 30min | -$180M | 期货恐慌平仓 |
| 现货 CVD 30min | -$15M | 现货仅小幅下跌 |
| 背离比例 | 12:1（合约 vs 现货） | 极端背离 |

**判断：** 期货端恐慌性平仓，不是真实现货卖出。FR -0.0061 触发 V1 做多，但等待清算密集区入场（$65,300-$66,000），不追市价。

**教训：** 宏观恐慌制造的期货卖压可与现货产生巨大背离，背离本身就是反弹信号，但要等清算密集区支撑才入场。

## 监测频率计划（GojiPower 官方规则）

### 三级自适应监测周期

**第一级：默认模式（无持仓）**
- 监测周期：**8 小时**
- 升级触发条件（任一）：
  - 价格变化 > 2%
  - 任一指标与前值发生较大变化
  - 资金费率触及阈值（V1 边缘）
- 升级后：切换为 **1 小时**，继续恶化则递进到 **30 分钟**

**第二级：美股开盘时段**
- 开始时间：7:30 UTC-4（美东时间）/ 4:30 AM PT
- 到美股收盘结束
- 监测周期：**1 小时**
- 升级触发：同上触发条件 → 递进到 **30 分钟**

**第三级：持仓中**
- 监测周期：**1 小时**
- 升级触发：同上触发条件 → **30 分钟**

### 监测逻辑決策树

```
当前是否有持仓？
├─ 有 → 每小时监测，有变化则 30 分钟
└─ 无 → 每 8 小时监测
        美股开盘时段？
        ├─ 是 → 升级为每小时
        └─ 否 → 保持 8 小时

每次监测后，是否触发升级条件？
├─ 是 → 下次监测缩短（最短 30 分钟）
└─ 否 → 保持当前周期
```

### Step 2：判断框架

拿到五指标数据后，**按以下逻辑判断**：

**先判断 V1（优先级最高）：**
```
如果 |FR| ≥ 0.0040：
  → 进入 V1 模式
  → 检查五指标中有几个方向一致
  → 3/5 以上一致 → V1 信号触发
  → 不足 3/5 → V1 边缘，继续观察

如果 |FR| < 0.0040：
  → 跳过 V1，进入 V2 判断
```

**V2 评分（FR 在 ±0.0010 内时）：**
```
现货 CVD 方向：向上 +3 / 向下 -3 / 横盘 0
OI 方向：增加 +1 / 减少 -1 / 横盘 0
订单簿：买墙 +1 / 卖墙 -1 / 平衡 0
清算流动性：下方密集（空头清算）+2 / 上方密集（多头清算）-2 / 均衡 0

总分 ≥ +5 → 做多
总分 ≤ -5 → 做空
-4 到 +4 → 观望
```

**强制观望条件（任一满足即不做判断）：**
- 重大宏观事件当天（Trump 讲话、央行政策、地缘政治）
- 节假日低流动性（Good Friday、圣诞、春节）
- 五指标方向互相矛盾，无法形成 3/5 共识
- 数据不完整（3 个以上指标 N/A）
